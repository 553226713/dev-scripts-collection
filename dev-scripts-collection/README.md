# Dev Scripts Collection 🚀

A collection of useful shell scripts and batch files for developers.

## 📦 Included Scripts

### Bash Scripts (Linux/macOS/Windows WSL)

| Script | Description |
|--------|-------------|
| `git-cleanup.sh` | Remove merged branches and stale remotes |
| `new-branch.sh` | Create and push a new branch |
| `qc.sh` | Quick commit - add, commit, push in one command |

### Windows Scripts

| Script | Description |
|--------|-------------|
| `setup-node.bat` | Quick Node.js project setup wizard |

## 🚀 Quick Start

### Linux/macOS

```bash
# Make scripts executable
chmod +x git-cleanup.sh
chmod +x new-branch.sh
chmod +x qc.sh

# Run scripts
./git-cleanup.sh
./new-branch.sh feature/my-feature
./qc "Fix: resolve login issue"
```

### Windows

```batch
# Run scripts
setup-node.bat
```
